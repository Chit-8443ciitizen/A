Xem kết quả ở cửa sổ console.log nhe

const axios = require('axios')
//1
const fetchUrls = async(urls) =>{
const results1 = []
for (const url of urls){
const res = await axios.get(url)
results1.push(res)
} return results1;
}
// 2
const fetchUrlsParallel = async(urls)=>{
const results2 = await Promise.all(
urls.map(function(url){
return axios.get(url);
})
);
return results2;
}

Đoạn code đầu tiên (fetchUrls) sử dụng một vòng lặp for...of kết hợp với từ khóa await để gửi các yêu cầu HTTP tuần tự. Đối với mỗi URL trong mảng urls, nó gửi một yêu cầu HTTP sử dụng axios.get và chờ đợi cho đến khi yêu cầu hoàn thành. Kết quả được trả về từ mỗi yêu cầu được đẩy vào mảng results1. Cuối cùng, mảng results1 được trả về khi tất cả các yêu cầu đã hoàn thành.

Đoạn code thứ hai (fetchUrlsParallel) sử dụng phương pháp Promise.all để gửi các yêu cầu HTTP đồng thời. Thay vì sử dụng vòng lặp, nó sử dụng phương thức map để tạo một mảng mới chứa các promise trả về từ các yêu cầu HTTP. Sau đó, Promise.all được sử dụng để chờ tất cả các promise trong mảng hoàn thành. Khi tất cả các yêu cầu đã hoàn thành, Promise.all trả về một mảng chứa kết quả của tất cả các yêu cầu HTTP.
